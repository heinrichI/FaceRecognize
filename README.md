# Face Recognition App

Веб-приложение для распознавания лиц с кластеризацией неизвестных людей.

## Возможности

- **Добавление известных лиц** - загрузка фотографий с именами в базу
- **Сканирование директорий** - поиск лиц на фото в указанной папке
- **Распознавание** - идентификация известных людей (cosine similarity)
- **Кластеризация** - автоматическое группирование неизвестных лиц (DBSCAN)

## Технологический стек

| Компонент | Технология |
|-----------|-------------|
| Язык | Python 3.11 |
| Веб-фреймворк | FastAPI |
| Распознавание лиц | DeepFace (ArcFace) |
| Векторная БД | Qdrant |
| Кластеризация | scikit-learn (DBSCAN) |
| Фронтенд | HTML + TailwindCSS |

## Быстрый старт

### Требования

- Python 3.11+
- Docker Desktop
- 8GB RAM (рекомендуется)

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repo-url>
cd FaceRecognize
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите Qdrant:
```bash
docker-compose up -d
```

4. Запустите приложение:
```bash
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

5. Откройте http://localhost:8000

## Использование

### 1. Добавление известных лиц

1. Введите имя человека
2. Загрузите несколько фото (jpg, png)
3. Нажмите "Добавить в базу"

### 2. Сканирование директории

1. Введите путь к папке с фото
2. Нажмите "Сканировать"
3. Дождитесь завершения

### 3. Просмотр результатов

- **Известные лица** - найденные в базе
- **Новые люди** - кластеры неизвестных лиц

## API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| GET | / | Главная страница |
| GET | /results | Результаты |
| GET | /api/health | Статус системы |
| POST | /api/known/add | Добавить известные лица |
| POST | /api/scan | Сканировать директорию |

## Алгоритм работы

```
┌─────────────────────────────────────────────────────────┐
│              Этап 1: Индексация (Enrollment)             │
│  1. Загрузка фото известных людей                      │
│  2. Извлечение embeddings (ArcFace)                     │
│  3. Сохранение в Qdrant с именем                       │
└─────────────────────────────────────────────────────────┘
                          ▼
┌─────────────────────────────────────────────────────────┐
│              Этап 2: Сканирование (Matching)            │
│  1. Сканиование фото в директории                   │
│  2. Детекция лиц (RetinaFace)                          │
│  3. Извлечение embeddings                              │
│  4. Поиск совпадений в базе (cosine similarity)        │
└─────────────────────────────────────────────────────────┘
                          ▼
┌─────────────────────────────────────────────────────────┐
│              Этап 3: Кластеризация (Discovery)           │
│  1. Отбор неизвестных лиц                              │
│  2. Применение DBSCAN                                  │
│  3. Группировка по кластерам                           │
└─────────────────────────────────────────────────────────┘
```

## Параметры

| Параметр | Значение | Описание |
|----------|----------|----------|
| similarity_threshold | 0.6 | Порог распознавания |
| dbscan_eps | 0.4 | Радиус кластера |
| dbscan_min_samples | 3 | Мин. лиц в кластере |
| model_name | ArcFace | Модель распознавания |
| detector_backend | retinaface | Детектор лиц |

## Устранение проблем

### Ошибка: tf-keras not found
```bash
pip install tf-keras
```

### Qdrant не запускается
```bash
docker-compose down
docker-compose up -d
```

### Малодо лиц обнаружено

- Убедитесь, что лица видны на фото
- Попробуйте разные ракурсы
